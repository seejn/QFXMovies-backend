# QFXMovies-Backend


## Setup

### Clone the Repository
```
git clone https://github.com/seejn/QFXMovies-backend.git
```
### Install required packages
```
cd QFXMovies-backend
npm i
```
### Create .env
set required env variables
```
cp .env.sample .env
```

### Seed Database [OPTIONAL]
```
node src/seed/seed.js
```
### Run Server
```
npm run dev
```
